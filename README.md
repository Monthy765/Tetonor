# Tetonor
---
Juego de lógica matemática (Hecho con IA)

**El Tetonor consta de una cuadrícula superior (16 resultados) y una lista inferior (16 números).**

- Elige **dos números** de la lista inferior.
- Con esa pareja, realiza una **suma** y una **multiplicación** cuyos resultados (ambos) estén en la cuadrícula superior.
- Cada número de la lista inferior solo puede usarse una vez.
- En la lista inferior los números deberán estar ordenados de forma ascendente.
- En la lista inferior puede haber números repetidos.

Debes encontrar todos los números de la lista inferior de tal forma que a todos los números de la cuadrícula les corresponda una operación válida (suma o producto) de alguna pareja de la lista inferior.
